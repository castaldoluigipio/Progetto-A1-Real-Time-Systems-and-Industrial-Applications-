Per prima cosa ci si scarica la cartella del progetto llama.cpp da GitHub:

git clone https://github.com/ggerganov/llama.cpp.git
cd llama.cpp

Per creare l'eseguibile dentro la cartella build/bin/, si usa CMake:

# 1. Configura l'ambiente di build
cmake -B build

# 2. Compila il codice in modalità Release
cmake --build build --config Release -j$(nproc)

Rimanendo dentro la directory principale llama.cpp, si crea la cartella models, ci si entra e si scaricano i 4 modelli di Qwen2.5 tramite wget:

mkdir -p models
cd models

wget https://huggingface.co/Qwen/Qwen2.5-0.5B-Instruct-GGUF/resolve/main/qwen2.5-0.5b-instruct-q4_k_m.gguf

wget https://huggingface.co/Qwen/Qwen2.5-1.5B-Instruct-GGUF/resolve/main/qwen2.5-1.5b-instruct-q4_k_m.gguf

wget https://huggingface.co/Qwen/Qwen2.5-3B-Instruct-GGUF/resolve/main/qwen2.5-3b-instruct-q4_k_m.gguf

wget https://huggingface.co/Qwen/Qwen2.5-7B-Instruct-GGUF/resolve/main/qwen2.5-7b-instruct-q4_k_m.gguf
